## Шаг A — собрать игру (25–30 мин)

### 1) Фон

Сцена → Фон → Загрузить `img/Фон.png`.

### 2) Переменные (для всех)

- `Монеты`  
- `монеты за клик` (= 1)  
- `монеты в секунду` (= 0)  

Показать на сцене.

### 3) Герой

1. Загрузить спрайт `img/Главный герой.png`, имя **Герой**.  
2. Ластиком убрать чёрный фон при необходимости.  
3. Код:

```
когда флаг нажат
  задать [Монеты] = 0
  задать [монеты за клик] = 1
  задать [монеты в секунду] = 0

когда этот спрайт нажат
  изменить [Монеты] на (монеты за клик)
```

### 4) Доход раз в секунду (герой или сцена)

```
когда флаг нажат
  повторять всегда
    ждать (1) секунду
    изменить [Монеты] на (монеты в секунду)
```

### 5) Кнопка50

Спрайт: серая → костюм1, коричневая → костюм2. Имя **Кнопка50**.

```
когда этот спрайт нажат
  если <(Монеты) ≥ (50)> то
    изменить [Монеты] на (-50)
    изменить [монеты за клик] на (1)
    следующий костюм
    ждать (0.2)
    переключить костюм на [серая]
```

### 6) Кнопка100

Аналогично, порог **100**, меняем `монеты в секунду` на **+1**.

Проверка: клики → покупки работают.

```
когда флаг нажат
  показаться
  перейти в x: (100) y: (80)

когда этот спрайт нажат
  изменить [Монеты] на (10)
  спрятаться
```
