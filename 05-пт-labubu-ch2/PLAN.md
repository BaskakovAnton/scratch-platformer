## Идея дня

«Вчера один этаж. Сегодня два. На втором — ловушки. Потом картинка Победа и стартовый экран.»

Порядок: **базовый (тест + ур.2) → средний (звуки/ловушки) → уверенный (меню)**.

---

## Шаг B — Базовый (30–40 мин)

### Стены — костюм 2

1. Спрайт **Стены** → загрузить `img/Лабиринт2.png` как костюм `лабиринт2`.  
2. Если касание цвета `#010101` не срабатывает на новом рисунке — дорисовать чёрные стены или оставить один макет + ловушки (честно в сдаче).

### Сообщения

Создать: `Уровень 2`, `Победа`, `Проигрыш`.  
Переменная: `номер уровня`.

### Логика

- Уровень 1: касание **Финиш** → `передать [Уровень 2]` (не победа).  
- На `Уровень 2`: стены → костюм 2; Лабубу → новый старт x,y; `номер уровня = 2`.  
- Уровень 2 + касание Финиш → `передать [Победа]`.

### Экран победы

Спрайт из `img/Экран победы.png`:

```
когда флаг нажат → спрятаться
когда я получу [Победа] → показаться → передний план → остановить все
```

### Чеклист тестера

1. Старт ур.1  
2. Финиш → ур.2 (стены сменились)  
3. Пройти ур.2  
4. Финиш → экран победы  

---

## Шаг C — Средний (20 мин)

1. Спрайты `img/Ловушка.png`, `img/Враг.png` на уровне 2.  
2. Касание → звук «ой» → `передать [Проигрыш]`.  
3. Спрайт из `img/Экран окончания игры.png` на Проигрыш.  
4. На Победу — победный звук.  
5. Проверить оба случая специально.

---

## Шаг D — Уверенный (15–20 мин)

Спрайт **Меню**: название + краткая инструкция (стрелки, флаг, ловушки).  
При флаге — меню сверху; по пробелу — спрятать меню и играть.
