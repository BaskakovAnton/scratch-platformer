# Flappy Буся

Игра в Scratch: Буся летит по коридору и проскакивает между когтеточками.

## Как открыть

1. Установи [Scratch Desktop](https://scratch.mit.edu/download) **или** открой [scratch.mit.edu](https://scratch.mit.edu) → **Создать**.
2. **Файл → Загрузить с компьютера** → выбери `busya.sb3`.
3. Нажми зелёный флаг → кнопку **«Полетели-и-и!»**.
4. Управление: **пробел** — взлететь.

## Файлы

| Путь | Что это |
|------|---------|
| `busya.sb3` | Готовая игра (релиз) |
| `img/backgrounds/` | Фон коридора |
| `img/busya/` | Костюмы Буси |
| `img/menu/` | ТВ и кнопка меню |
| `img/world/` | Когтеточка |
| `img/references/` | Референсы для картинок |
