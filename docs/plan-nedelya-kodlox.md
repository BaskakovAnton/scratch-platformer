# План недели: Кодлокс «Герой» с дочерью

Общий план занятий. Учитель — папа, ученик — дочь.  
Курс: **Герой. Практические и домашние задания** (Scratch).  
Охват: уроки **1 → 1-5** (включая Испытание Лабубу ч.2).

Связанный релиз игры Буси: `busya.sb3`, ассеты в `img/`.

---

## Видение

### Формула дня

1. **Играем 3–5 мин** — «что это за игра?»
2. **Разбираем устройство** — 2–4 «кирпичика»
3. **Делаем ДЗ уровнями** — базовый → средний → уверенный
4. **Сдаём** — скрин в LMS, короткое «что умеет игра»

### Роли

| Учитель (папа) | Ученик (дочь) |
|----------------|---------------|
| Объясняет правило одним предложением | Выбирает вид, тему, звук, название |
| Показывает *где* искать блоки | Сама таскает блоки / меняет костюмы |
| Чинит, когда «сломалось» | Тестирует зелёным флагом |
| Говорит «что проверить» | Говорит «готово» своими словами |

Правило: **один новый смысл за раз**.

### Фразы-якоря

- «Зелёный флаг — проверка.»
- «Один блок. Только один.»
- «Сломалось? Что видишь на сцене?»
- «Ты художник, я инженер.»

### Чего не делаем

- Не гоним все задания ценой усталости
- Видео LMS не обязательны — учитель объясняет суть
- Не пугаем терминами: вместо «переменная / клон / broadcast» → **коробочка с числом / копия / крик «Уровень 2!»**
- Не правим код за неё молча — сначала она пробует

### Аварийный минимум дня

Только **базовый** уровень + сдача. Остальное — в буфер пятницы или следующую неделю.

# Карточка 1 · Понедельник  
## Урок 1. Подготовительный (Iron Man)

### Цель дня

«Сегодня мы не новую игру изобретаем. У нас уже есть железный герой — мы его **доделаем и починим**, как мастерскую.»

### Время

40–50 мин

### Что открыть

Проект Iron Man из курса. LMS → ДЗ урока 1.

### Объяснение (скрипт)

**Учитель:** Представь: игру уже почти собрали на заводе, но забыли пару винтиков. Мы — мастерская.  
В Scratch три места: **сцена** (что видим), **спрайты** (кто в игре), **блоки** (правила).  
Сегодня: смотрим, что доделать → делаем → **зелёный флаг** → проверяем.  

**Ученик:** открывает проект, играет.  
**Учитель:** Кто герой? Что делать? Когда конец?

### Кирпичики

Герой · Управление · Конец игры / цель

### ДЗ (одно)

1. Играем 2 минуты — запоминаем, как живёт игра.  
2. Доделываем то, чего не хватает.  
3. Полный прогон: старт → игра → конец.  
4. Скрин/фото → «Выполнить задание».

### Если сломалось

- Не двигается → есть ли реакция на клавишу/пробел  
- Не видно → размер, не спрятан ли  
- Улетел → «перейти в x y» при зелёном флаге  

### Финиш

«Урок один закрыт. Завтра — летающая игра.»

# Карточка 2 · Вторник  
## Урок 1-2. Капитан Взлёт (можно Буся)

### Цель дня

«Сделаем свою леталку: красивую, с нужной сложностью и с обложкой.»

### Время

55–70 мин

### Что открыть

Проект «Капитан Взлёт» / `busya.sb3`.

### Объяснение (скрипт)

**Учитель:** Герой слева. Пробел — вверх. Без пробела — падает. Справа едут трубы/когтеточки.  
Три слова: **тяжесть**, **прыжок**, **препятствие**.  
Сегодня: наряжаем → сложность → обложка.

**Ученик:** играет 1–2 минуты.  
**Учитель:** Где дырка? Что если врезаться?

### Кирпичики

Герой · Препятствие · Счёт · Скорость

### Базовый (~20 мин)

Меняем только картинки: **фон → герой → трубы**.  
Зелёный флаг: всё ещё летает?

### Средний (~20 мин)

Крутим: как часто появляется труба (`ждать`) и как быстро едет (`изменить x`).  
Выбрать сложность «интересную для тебя».

### Уверенный (~15–20 мин)

Название + обложка (Scratch или редактор).  
Сдать все три уровня.

### Если сломалось

- Огромный герой → размер 30–45%  
- Трубы стоят → клоны / флаг заново  
- Сразу смерть → костюм слишком большой  

### Финиш

«У тебя игра с лицом и характером. Завтра — монетки.»

# Карточка 3 · Среда  
## Урок 1-3. Очень странные дела (кликер)

### Цель дня

«Кликаешь — копишь монеты — покупаешь силу. Добавим свой секретный подарок.»

### Время

55–65 мин

### Что открыть

Проект кликера из курса.

### Объяснение (скрипт)

**Учитель:** Это **магазин силы**. Клик → монета. Улучшения: больше за клик или каплет само.  
Коробочки: **Монеты**, **За клик**, **В секунду**.

**Ученик:** кликает минуту.  
**Учитель:** Сколько монет? Что делает кнопка?

### Кирпичики

Герой · Монеты · Кнопки улучшений · Бонус

### Базовый (~15–20 мин)

Новый фон + размер героя. Клик и кнопки работают?

### Средний (~15 мин)

Звук на кнопку улучшения. Короткий, смешной. Нажать — услышать.

### Уверенный (~20–25 мин)

Новый спрайт-бонус: касание/клик → монеты +N → спрятаться.  
Проверка: было 5 → стало больше.

### Если сломалось

- Монеты в минус → нет проверки «хватает ли»  
- Бонус не пропадает → нет «спрятаться»  
- Звук не там → не тот спрайт  

### Финиш

«Ты изобрела подарок. Завтра — лабиринт Лабубу.»

# Карточка 4 · Четверг  
## Урок 1-4. Испытание Лабубу ч.1

### Цель дня

«Лабиринт станет твоим: всё видно, и есть ещё один бонус для сбора.»

### Время

55–70 мин

### Что открыть

Проект Лабубу ч.1.

### Объяснение (скрипт)

**Учитель:** Ходим, не летаем. Стрелки — движение. Стены — нельзя сквозь. Звёзды — собирать. Флажок — финиш.  
Если плохо видно — игрок злится. Сегодня глаз сразу находит героя, стены и цель.

**Ученик:** проходит кусок.  
**Учитель:** Где старт? Финиш? Сколько звёзд?

### Кирпичики

Лабубу · Стены · Звёзды · Флажок · Новый бонус

### Базовый (~20–25 мин)

Фон / стены / флажок. Проверка пальцем: герой — стена — звезда — флажок. Пройти до флага.

### Средний (~20–25 мин)

Ещё бонус в 2–3 местах. Касание → счёт → спрятать. Собрать бонусы + флаг.

### Уверенный (~15–20 мин)

Один мир (космос, лес, замок…). Фон, стены, флажок, звёзды в одном стиле.

### Если сломалось

- Сквозь стены → нет отталкивания  
- Звезда не собирается → не тот спрайт  
- Плохо видно → контраст  

### Финиш

«Завтра — второй уровень, ловушки и экран „Победа!“»

# Карточка 5 · Пятница  
## Урок 1-5. Испытание Лабубу ч.2

### Цель дня

«Настоящая игра: два уровня, опасности, звуки и стартовый экран с названием.»

### Время

60–75 мин  
*(минимум — только базовый полный прогон)*

### Что открыть

Проект Лабубу **ч.2**.

### Объяснение (скрипт)

**Учитель:** Вчера один лабиринт. Сегодня — **главы**. Уровень 1 → уровень 2 → **«Победа!»**  
Ловушка/враг — нельзя трогать.  
Три роли сегодня: тестировщик → звукорежиссёр → режиссёр афиши.

**Порядок жёсткий:** сначала базовый (игра живая), потом звуки, потом стартовый экран.

### Кирпичики

Уровень 1 / 2 · Ловушка/враг · «Победа!» · Звуки · Стартовый экран

### Базовый (~25 мин) — главный

Чеклист вслух:

1. Старт уровня 1  
2. Переход на уровень 2  
3. Пройти уровень 2  
4. Флаг  
5. Экран **«Победа!»**

Баг: «что сделала → что увидела → как должно». Чинить по одному.

### Средний (~15–20 мин)

1. Ловушка/враг → звук проигрыша  
2. Победа / 2-й уровень → звук победы  

Проверить оба случая специально.

### Уверенный (~20–25 мин)

Название + стартовый экран:

- как двигать  
- цель (флаг)  
- чего бояться  
- как начать  

Зелёный флаг → сначала экран → старт уровня 1.

### Если сломалось

- Нет 2 уровня → нет смены/сообщения  
- Победа рано → условие слишком широкое  
- Экран не уходит → нет «спрятаться»  

### Финиш недели

«Пять дней — пять игр. Ты умеешь чинить, летать, копить, ходить по лабиринту и делать игру с двумя уровнями.»

### Порядок работы с `.sb3` из `DZ/`

1. Scratch → **Файл → Загрузить с компьютера** → выбрать файл из `DZ/`.  
2. Сразу **Файл → Сохранить на компьютер** под новым именем (`labubu-ch1-yana.sb3`), оригинал в `DZ/` не портить.  
3. С дочерью: сначала поиграть 2–3 мин, потом ДЗ по карточке дня.  
4. В LMS сдавать скрин **вашей** сохранённой копии.

### Порядок работы с PNG

1. Спрайт → **Костюмы → Загрузить** нужный PNG.  
2. Чёрный фон у героя/кнопок/врага — вырезать ластиком в Scratch или заранее прозрачность.  
3. Кнопки кликера: серая = обычная, коричневая = «нажата / куплено» (второй костюм).  
4. Фон сцены → **Сцена → Фон → Загрузить**.

### Чего в `DZ/` пока нет

- Готового `.sb3` **кликера** (только PNG в `Заготовка/`) — проект с урока/LMS, картинки подставляете вы.  
- Отдельного готового `.sb3` именно **«Лабубу ч.2»** с двумя уровнями — есть ч.1 (`Испытание Лабубу.sb3`) + картинки для достройки ч.2. Если скачаете полный проект ч.2 с LMS — положите рядом в `DZ/лабубу/`.
