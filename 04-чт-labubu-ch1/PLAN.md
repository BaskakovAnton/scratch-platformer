## Как устроена игра

- Стрелки: шаг **5** по x/y  
- Стена = цвет `#010101` → откат шага  
- 4 звезды → `Звёзды +1`, спрятаться  
- Касание **Финиш** → «Победа!» → стоп  
- Старт героя: x=-20 y=159 · Финиш: x=22 y=-164  

---

## Шаг A — поиграть (5 мин)

Пройти лабиринт, собрать звёзды, финиш.  
«Что нельзя? Что собирать? Где победа?»

---

## Шаг B — Базовый: видно и красиво (20 мин)

1. Сцена → фон (галерея) при желании.  
2. **Спрайт Лабубу** → Костюмы → загрузить `img/Sprite-Labubu.png` (стереть чёрный фон).  
3. Размер — чтобы проходил коридоры.  
4. **Финиш** — можно флажок из галереи.  
5. Пройти до «Победа!». Всё должно быть хорошо видно.

---

## Шаг C — Средний: ещё бонус (20–25 мин)

1. Дублировать звезду **или** новый спрайт → имя **Бонус1**.  
2. Код как у звезды: касание Лабубу → `Звёзды +1` → звук → спрятаться.  
3. **Удалить** у бонуса строку `задать Звёзды = 0` (сброс только в одном месте — у Звезда1).  
4. Поставить 2–3 бонуса в разных x,y.  
5. Пройти: собрать всё + финиш.

---

## Шаг D — Уверенный: один стиль (15 мин)

Один мир (лес/космос/сладкий…): фон + Лабубу + финиш + бонусы. Скрин.
